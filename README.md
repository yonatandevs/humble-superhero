# Humble Superhero APP

## Overview

This is a full-stack application built with NestJS (backend) and React (frontend). It allows users to add and view superheroes sorted by their humility score.

## Features

- Add a superhero with name, superpower, and humility score.
- View superheroes sorted by humility score.
- Validation for humility score (1-10).

## Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   ```

## API Endpoints

- **POST /superheroes**: Add a new superhero.
- **GET /superheroes**: Fetch all superheroes sorted by humility score.
