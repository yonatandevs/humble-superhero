# Humble Superhero API

## Overview

This API allows users to add and retrieve superheroes, sorted by their humility score. It’s built with NestJS and uses an in-memory database.

## Features

- Add a superhero with name, superpower, and humility score.
- Fetch superheroes sorted by humility score.
- Validation for humility score (1-10).

## Setup

1. Clone the repository.
2. Run `npm install`.
3. Start the server with `npm run start`.

## API Endpoints

- **POST /superheroes**: Add a new superhero.
- **GET /superheroes**: Fetch all superheroes sorted by humility score.
